# One Heart Productions - Project Guidelines

## Design Principles (Sticky)
- **Aesthetic:** "Dark Luxe" & "Ultra-Elevation". Premium, theatrical, and international.
- **Colors:** Background: `#000000` (Pure Black). Accents: `#f0eee9` (Pearl). Glows: `rgba(226, 232, 240, 0.1)` (Pearl Mist).
- **Typography:** Headings: "Clash Display". Body: "Outfit" / "IBM Plex Sans Thai".
- **Ultra-Elevation:** Use 3D scaling effects (0.8x to 1.0x) on main containers when menus open. Apply deep multi-layered shadows and razor-thin rim highlights.
- **Quiet Luxury:** High-impact visuals, minimal UI elements, sophisticated spacing.
- **Anti-Patterns:** Avoid "basic", "cheap", or "generic" designs. No light mode.

## Development Rules
- **Tech Stack:** SvelteKit 5 (Runes), Tailwind 4, Lenis (Smooth Scroll).
- **Communication:** Respond in Thai (as per user preference). Be concise and prioritize iterative polish.
- **Verification:** Always verify UI changes against the "Dark Luxe" aesthetic.
